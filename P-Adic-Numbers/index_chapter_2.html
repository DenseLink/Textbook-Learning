Quick Recap: Why p-adic Numbers?

In everyday math, we measure "closeness" of numbers using the standard absolute value: 0.001 is close to 0, and 1000000 is far away. But what if we flipped this intuition? In p-adic math, numbers divisible by high powers of a prime p are considered small. So for the 5-adic absolute value, 510 = 9,765,625 is incredibly tiny — almost zero! This chapter makes that idea rigorous.

§2.1 Absolute Values on a Field

💡 In Plain English

We all have an instinct for the "size" of a number — how far it is from zero. This section steps back and asks: what are the bare rules that any sensible notion of size must obey? The answer is short: a size is never negative (and only zero is zero-sized), the size of a product is the product of the sizes, and the size of a sum is no bigger than the sum of the sizes (the "triangle inequality" — no shortcuts by adding).

Anything obeying those three rules earns the name absolute value. The surprise of the whole book is that the familiar one isn't the only option: on the rational numbers there are entirely different, equally valid "size" rules — the p-adic ones — where being divisible by p makes a number small. Setting up the rules abstractly is what lets these exotic measuring sticks join the club.

📘 Definition 2.1.1 — Absolute Value on a Field

Formal: An absolute value on a field 𝕜 is a function |·| : 𝕜 → ℝ₊ satisfying:
(i) |x| = 0 ⟺ x = 0  (ii) |xy| = |x|·|y|  (iii) |x + y| ≤ |x| + |y|

It is non-archimedean if it also satisfies: (iv) |x + y| ≤ max{|x|, |y|}.

Plain-language: An absolute value is just a "size" function that tells you how big a field element is. Think of it as a ruler. The three rules say: (1) only zero has size zero, (2) the size of a product equals the product of sizes, and (3) the triangle inequality — going from A to C directly is never longer than going A→B→C.

A non-archimedean absolute value replaces the triangle inequality with a stronger version: the size of a sum is at most the maximum of the individual sizes. This seemingly small change leads to wildly different geometry!

🔍 Real-World Example

Hierarchical file system: Think of files in nested folders. The "distance" between two files could be measured by how far up the folder tree you need to go to find a common ancestor. /docs/math/hw1.tex and /docs/math/hw2.tex are "close" (same folder), while /docs/math/hw1.tex and /photos/cat.jpg are "far." This distance satisfies the ultrametric inequality — the distance between any two files is at most the maximum of their distances to a third file.

DNA sequences: Two gene sequences sharing a more recent common ancestor are "closer" in evolutionary distance — another natural ultrametric.

📘 Definition 2.1.2 — The p-adic Valuation

Formal: Fix a prime p. For a nonzero integer n, vp(n) is the exponent of the highest power of p dividing n. For rationals: vp(a/b) = vp(a) − vp(b). By convention, vp(0) = +∞.

Plain-language: The p-adic valuation simply counts how many times the prime p divides a number. Think of it as asking: "how p-ish is this number?"

  • v₅(250) = ? → 250 = 5² × 10 = 5² × 2 × 5 = 5³ × 2, so v₅(250) = 3 (wait, 250 = 2·125 = 2·5³, yes, so v₅(250)=3)
  • v₂(12) = ? → 12 = 4 × 3 = 2² × 3, so v₂(12) = 2
  • v₃(7) = 0 (3 doesn't divide 7 at all)
🔍 Real-World Example: Quality Control

Imagine a factory producing items in batches of p = 10. An order number like 5000 is "very round" — it's divisible by 10³. The valuation v₁₀(5000) = 3 tells us this order is at a "milestone." Order 5001 has v₁₀(5001) = 0 — not special at all. The p-adic valuation measures this kind of "roundness" or "divisibility depth."

📙 Lemma 2.1.3 — Properties of vp

Formal: For all x, y ∈ ℚ: (i) vp(xy) = vp(x) + vp(y), and (ii) vp(x + y) ≥ min{vp(x), vp(y)}.

Step-by-step reasoning:

Part (i): If x = pa·x' and y = pb·y' (where p doesn't divide x' or y'), then xy = pa+b·x'y'. Since p doesn't divide x'y' (p is prime!), we get vp(xy) = a + b = vp(x) + vp(y). Multiplying numbers adds their p-adic "depth."
Part (ii): Factor out the smaller power: if a ≤ b, then x + y = pa(x' + pb−ay'). The quantity in parentheses might have additional factors of p, so vp(x + y) ≥ a = min(a, b). Adding can only increase (or maintain) the minimum p-divisibility.
📘 Definition 2.1.4 — The p-adic Absolute Value

Formal: |x|p = p−vp(x) for x ≠ 0, and |0|p = 0.

Plain-language: The more times p divides a number, the smaller its p-adic absolute value. It's the inverse of our usual intuition! If p = 5:

  • |25|₅ = 5−2 = 1/25 (very small — 25 is very "5-ish")
  • |3|₅ = 50 = 1 (not divisible by 5 at all, so it's "far from zero")
  • |510|₅ = 5−10 ≈ 0.0000001 (huge number, but tiny p-adically!)

📗 Proposition 2.1.5 — |·|p is Non-Archimedean

The p-adic absolute value satisfies all four axioms of Definition 2.1.1, including the ultrametric inequality |x + y|p ≤ max{|x|p, |y|p}.

Proof sketch

This follows directly from Lemma 2.1.3. The key translations are:

Multiplicativity: |xy|p = p−vp(xy) = p−vp(x)−vp(y) = |x|p·|y|p ✓
Ultrametric: vp(x+y) ≥ min{vp(x), vp(y)} means −vp(x+y) ≤ max{−vp(x), −vp(y)}, so p−vp(x+y) ≤ max{p−vp(x), p−vp(y)} ✓
🔍 Example: The Product Formula

A beautiful fact: for any nonzero rational number x, the product of all its absolute values equals 1:

|x|∞ · ∏p prime |x|p = 1

For x = 12: |12|∞ = 12, |12|₂ = 1/4, |12|₃ = 1/3, and |12|p = 1 for p ≥ 5. So 12 × (1/4) × (1/3) = 1. ✓

§2.2 Basic Properties

💡 In Plain English

Once you have the three rules from the previous section, a bunch of "obvious" facts follow automatically — and it's worth checking they really do flow from the rules rather than from our real-number habits. For example, the size of 1 must be exactly 1, the size of −1 equals the size of 1, and flipping a number's sign never changes its size.

A cute consequence: any number that's a root of unity (some power of it equals 1) must have size exactly 1, which forces a striking fact — on a finite field the only possible absolute value is the boring "trivial" one (everything nonzero has size 1). These little lemmas are the basic toolkit we'll lean on constantly later.

📙 Lemma 2.2.1 — Basic Properties of Absolute Values

Formal: For any absolute value |·| on any field 𝕜: (i) |1| = 1, (ii) xⁿ = 1 ⟹ |x| = 1, (iii) |−1| = 1, (iv) |−x| = |x|, (v) finite fields only have the trivial absolute value.

Step-by-step:

(i) |1| = 1: Since 1·1 = 1, the multiplicativity axiom gives |1|² = |1|. The only positive real solution is |1| = 1.
(iii) |−1| = 1: Since (−1)² = 1, we get |−1|² = |1| = 1, so |−1| = 1.
(iv) |−x| = |x|: |−x| = |−1|·|x| = 1·|x| = |x|.
(v) Finite fields: In a finite field with q elements, every nonzero element satisfies xq−1 = 1 (Fermat's little theorem generalized). By (ii), |x| = 1 for all nonzero x.

Intuition: These are the "obvious" properties you'd want any notion of size to satisfy. Negating doesn't change size, and the multiplicative identity has size 1.

🔍 Real-World Example: Modular Arithmetic

Consider a clock with 12 hours. The "size" of a time is meaningless in the modular sense — going forward 5 hours or backward 7 hours lands on the same spot. This is why finite fields (which have a similar cyclic structure) can only support the trivial absolute value: all nonzero elements are "equally big" because they cycle back to 1.

📙 Lemma 2.2.2 — Equivalent Condition for Non-Archimedean

Formal: |x + y| ≤ max{|x|, |y|} for all x, y ⟺ |z + 1| ≤ max{|z|, 1} for all z.

Why this matters: To check if an absolute value is non-archimedean, you don't need to test every pair (x, y). It suffices to test just expressions of the form z + 1. This is a massive simplification!

Proof idea
⇒ direction: Just set x = z, y = 1 in the ultrametric inequality.
⇐ direction: For any x, y with y ≠ 0, set z = x/y. Then |x/y + 1| ≤ max{|x/y|, 1}. Multiply both sides by |y| to get |x + y| ≤ max{|x|, |y|}. ✓
📙 Lemma 2.2.3 — Sufficient Conditions for Non-Archimedean

Formal: If |·| satisfies (i) |x|=0 ⟺ x=0, (ii) |xy|=|x||y|, and (iii) |x|≤1 ⟹ |x−1|≤1, then |·| is a non-archimedean absolute value.

Plain-language: You don't even need to verify the triangle inequality directly! If elements inside the "unit ball" {x : |x| ≤ 1} stay inside when you subtract 1, the function is automatically a non-archimedean absolute value. The unit ball is "closed under shifting by 1" — and that's enough!

📗 Theorem 2.2.4 — Characterization via the Image of ℤ

Statement: An absolute value |·| on a field 𝕜 is non-archimedean if and only if |a| ≤ 1 for every element a in the image of ℤ in 𝕜.

Plain-language: This is the big characterization theorem. It says: to test whether an absolute value is non-archimedean, just check if all "integers" inside the field have absolute value ≤ 1. If |n| ≤ 1 for every n ∈ ℤ (or its image), you're in non-archimedean territory.

Proof walkthrough
Easy direction (⇒): If |·| is non-archimedean, then by induction: |n| = |1+1+…+1| ≤ max{|1|,…,|1|} = 1.
Hard direction (⇐): Assume |n| ≤ 1 for all integers. Use the binomial theorem: |x+1|m = |∑ C(m,k)xk| ≤ ∑|C(m,k)||x|k ≤ ∑|x|k ≤ (m+1)·max{1,|x|}m. Take m-th roots: |x+1| ≤ (m+1)1/m·max{1,|x|}. Let m → ∞ (since (m+1)1/m → 1): |x+1| ≤ max{1,|x|}. ✓

The "hocus-pocus" is the limit trick — the inequality holds for every m, and the extra factor (m+1)1/m shrinks to 1.

🔍 Example: Testing the 3-adic absolute value

Is |·|₃ non-archimedean? Check: |1|₃ = 1, |2|₃ = 1, |3|₃ = 1/3, |4|₃ = 1, |5|₃ = 1, … Every integer has 3-adic absolute value ≤ 1 (it's either 1 or a negative power of 3). By Theorem 2.2.4, it's non-archimedean. ✓

Now the usual |·|: |1| = 1, |2| = 2, |3| = 3, … We have |2| = 2 > 1, so the usual absolute value is archimedean. ✓

§2.3 Topology

💡 In Plain English

As soon as you can measure size, you can measure distance: the distance between two numbers is just the size of their difference. With distance comes the whole language of topology — which points are "near" each other, what it means for a sequence to converge, and when a function is continuous (small input changes cause small output changes).

For non-archimedean (p-adic-style) absolute values this geometry gets delightfully weird. The triangle inequality is replaced by a stronger version, which leads to facts like "every triangle is isosceles" and "every point inside a ball is its center." Building this topological vocabulary now lets us later do genuine calculus and analysis in the p-adic world.

📘 Definition 2.3.1 & 2.3.2 — Metric and Continuity

Metric: d(x, y) = |x − y| defines the distance between elements.

Continuity: f is continuous at x₀ if nearby inputs produce nearby outputs: d(x, x₀) < δ ⟹ d(f(x), f(x₀)) < ε.

Plain-language: The absolute value turns our field into a "metric space" — a place where we can measure distances. Continuity means what you'd expect: small input changes produce small output changes. The p-adic metric measures distance in a fundamentally different way than the usual one.

🔍 Example: 5-adic distances

In the 5-adic world, d(3, 28) = |3 − 28|₅ = |−25|₅ = |25|₅ = 5−2 = 1/25. So 3 and 28 are very close 5-adically (they differ by 25 = 5²). Meanwhile, d(3, 4) = |−1|₅ = 1 — they're "far apart" because their difference isn't divisible by 5 at all!

📙 Lemma 2.3.3 — The Ultrametric Inequality

Statement: |·| is non-archimedean ⟺ d(x, y) ≤ max{d(x, z), d(z, y)} for all x, y, z.

Intuition: In an ultrametric space, the "detour" through z is always at least as long as the direct route. But the inequality is sharper than usual — instead of summing the two legs, we only need the maximum. This means taking a detour can never make the trip more than the longer leg!

📗 Proposition 2.3.4 — The "Strongest Wins" Principle

Statement: If |x| ≠ |y|, then |x + y| = max{|x|, |y|}.

Plain-language: When two numbers have different p-adic sizes, the sum's size equals the size of the bigger one — the "strongest wins." The smaller one is too small to affect the total. It's like adding a whisper to a shout — you still hear the shout.

Proof walkthrough
Step 1: Assume |x| > |y|. By the ultrametric inequality, |x + y| ≤ max{|x|, |y|} = |x|.
Step 2: Now write x = (x + y) − y. So |x| ≤ max{|x + y|, |y|}.
Step 3: Since |x| > |y|, the max must be |x + y|. So |x| ≤ |x + y|.
Conclusion: Combining: |x| ≤ |x + y| ≤ |x|, so |x + y| = |x| = max{|x|, |y|}. ✓
📗 Corollary 2.3.5 — All Triangles Are Isosceles!

In an ultrametric space, every triangle has (at least) two sides of equal length.

Why: Take vertices x, y, z. We have (x−y) + (y−z) = (x−z). If the first two sides have different lengths, "strongest wins" tells us the third side equals the longer one. Either way, two sides match!

📘 Definition 2.3.6 — Open and Closed Balls

Open ball: B(a, r) = {x ∈ 𝕜 : |x − a| < r}

Closed ball: B̄(a, r) = {x ∈ 𝕜 : |x − a| ≤ r}

Plain-language: Just like in normal geometry, a ball is all points within a certain distance of a center. The open ball excludes the boundary; the closed ball includes it.

📗 Proposition 2.3.7 — Mind-Bending Ball Properties

In a non-archimedean field:

  • (i)-(ii) Every point is a center: If b is inside a ball centered at a, then the ball centered at b with the same radius is the same ball.
  • (iii)-(iv) Balls are clopen: Both open and closed balls are simultaneously open and closed sets (with empty boundary!).
  • (v)-(vi) Balls are nested or disjoint: Two balls either don't overlap at all, or one contains the other entirely. No partial overlaps!

Intuition: Imagine bubbles that can't partially overlap — if they touch at all, one swallows the other. And you can stand anywhere inside a bubble and claim to be its center. This is radically different from Euclidean geometry!

📘 Definition 2.3.8 — Clopen Sets

A set is clopen if it is both open and closed simultaneously. In the usual real line, only ∅ and ℝ are clopen. But in p-adic spaces, every ball is clopen!

📗 Proposition 2.3.9 — Total Disconnectedness

Statement: In a field with a non-archimedean absolute value, the only connected subsets are single points.

Plain-language: The p-adic world is completely "shattered." You can't draw a continuous path between any two distinct points. Every set with more than one point can be split into two separate open pieces. It's like a Cantor set — infinitely fragmented.

🔍 Example: Why No Continuous ℝ → ℚp Functions

Corollary 2.3.10 says there are no non-constant continuous functions from ℝ to a p-adic field. Here's the intuition: ℝ is connected (one piece), and continuous functions send connected sets to connected sets. But ℚp is totally disconnected (shattered). The only connected subset of ℚp is a single point, so the image can only be one point — the function must be constant!

§2.4 Algebra

💡 In Plain English

This section looks at the algebraic shadow cast by a non-archimedean absolute value. The elements of size at most 1 form a tidy set called the valuation ring — you can add, subtract, and multiply within it and never escape. It's the abstract version of "the integers sitting inside the rationals."

Even better, this ring has a single special maximal ideal (the elements of size strictly less than 1), which makes it a local ring — a ring with exactly one "center of badness." That clean structure, with one prime to worry about, is precisely what makes p-adic arithmetic so much more orderly than ordinary integer arithmetic, where infinitely many primes all compete at once.

📗 Proposition 2.4.1 — The Valuation Ring Is a Local Ring

Statement: Let |·| be non-archimedean on 𝕜. Then:

  • 𝒪 = {x ∈ 𝕜 : |x| ≤ 1} is a subring (closed under +, ×, and contains 1)
  • 𝒫 = {x ∈ 𝕜 : |x| < 1} is a maximal ideal of 𝒪
  • Every element outside 𝒫 (but in 𝒪) is invertible in 𝒪

Plain-language: The "unit ball" 𝒪 = B̄(0,1) isn't just a geometric object — it has algebraic structure! It's a ring (you can add and multiply inside it). The "open unit ball" 𝒫 = B(0,1) is its unique maximal ideal. Elements on the "boundary" (|x| = 1) are invertible. This makes 𝒪 a local ring — a ring with a unique maximal ideal.

Proof sketch
𝒪 is a ring: If |x|, |y| ≤ 1, then |x + y| ≤ max{|x|,|y|} ≤ 1 and |xy| = |x||y| ≤ 1. Also |1| = 1 ≤ 1. ✓
𝒫 is an ideal: If |x| < 1 and |r| ≤ 1, then |rx| = |r||x| < 1. Also |x+y| ≤ max{|x|,|y|} < 1 if both are in 𝒫. ✓
Complement is units: If x ∈ 𝒪 \ 𝒫, then |x| = 1, so |1/x| = 1/|x| = 1, meaning 1/x ∈ 𝒪. ✓
𝒫 is maximal: Any ideal strictly containing 𝒫 would contain a unit, hence all of 𝒪. ✓
📘 Definition 2.4.2 — Valuation Ring, Ideal, Residue Field
  • Valuation ring: 𝒪 = {x ∈ 𝕜 : |x| ≤ 1}
  • Valuation ideal: 𝒫 = {x ∈ 𝕜 : |x| < 1}
  • Residue field: κ = 𝒪/𝒫 (the quotient — what's left when you "mod out" by the small elements)
📗 Proposition 2.4.3 — Explicit Description for ℚ with |·|p

For 𝕜 = ℚ with the p-adic absolute value:

  • Valuation ring: ℤ(p) = {a/b ∈ ℚ : p ∤ b} — fractions whose denominators aren't divisible by p
  • Valuation ideal: pℤ(p) = {a/b : p ∤ b, p | a} — fractions with an extra factor of p in the numerator
  • Residue field: 𝔽p — the finite field with p elements (integers mod p)
🔍 Example: The 5-adic Valuation Ring

For p = 5:

  • In 𝒪: 3/7 (denominator 7, not divisible by 5) ✓, 1/3 ✓, 2/11 ✓, 17 ✓
  • Not in 𝒪: 1/5 (denominator is 5) ✗, 3/25 (denominator 25 = 5²) ✗
  • In 𝒫: 5/7 (numerator has 5, denominator doesn't) ✓, 10/3 = (2·5)/3 ✓
  • Residue field: 𝔽₅ = {0, 1, 2, 3, 4} with arithmetic mod 5
🔍 Real-World Analogy: Localization

Think of ℤ(p) as a "magnifying glass" focused on the prime p. You allow all fractions except those requiring division by p. It's like a bank that deals in every currency except one: everything works normally until you try to use that forbidden currency.

The residue field 𝔽p is what you see when you "blur your vision" so that everything differing by a multiple of p looks the same. It's modular arithmetic — the simplest algebraic structure that emerges from the p-adic world.
