The Big Picture

In Chapter 2 we studied absolute values abstractly. Now we ask two concrete questions about ℚ:

  1. How many absolute values does ℚ have? Ostrowski's theorem gives a stunning answer: up to equivalence, exactly one for each prime p, plus the "usual" one at infinity. Nothing else exists.
  2. ℚ has holes — how do we fill them? Just as the real numbers ℝ complete ℚ with respect to the usual absolute value, we build a brand-new field ℚp that completes ℚ with respect to |·|p.

By the end you'll understand the famous slogan: "ℝ is just one of infinitely many completions of ℚ — one for every prime."

§3.1 Absolute Values on ℚ

We already know three kinds of absolute value on ℚ: the trivial one, the usual one |·|∞ (from ℝ), and for each prime p the p-adic one |·|p. It is convenient to treat ∞ as an "infinite prime" so we can write |·|p for all primes p ≤ ∞. The goal of this section: prove these are all of them (up to equivalence) and discover how they fit together via the product formula.

📘 Definition 3.1.1 — Equivalent Absolute Values

Formal: Two absolute values |·|1 and |·|2 on a field k are equivalent if they define the same topology on k — i.e., every set that is open with respect to one is also open with respect to the other.

Plain-language: Two "rulers" are equivalent if they can't be told apart by topology. They might assign different numbers as sizes, but they agree on which points are "near" which — same open sets, same convergent sequences, same notion of limit. Think of measuring in inches vs. centimeters: the numbers differ, but "close" means the same thing.

📒 Lemma 3.1.2 — Convergence via Open Sets Lemma

Statement: For a field k with absolute value |·|, the following are equivalent:

(i) limn→∞ xn = a;  (ii) any open set containing a contains all but finitely many of the xn.

Plain-language: Convergence can be described purely in terms of open sets, without ever mentioning specific distances. That's the key trick: if two absolute values have the same open sets, they automatically agree on every limit too.

Proof Idea

(ii)⇒(i): the open ball B(a,ε) is itself an open set, so all but finitely many xn land in it — that's exactly the ε-definition of a limit. (i)⇒(ii): given an open set U ∋ a, fit a ball B(a,r) ⊂ U; convergence pushes the tail of the sequence into that ball, hence into U.

📐 Proposition 3.1.3 — Four Faces of Equivalence Prop

Statement: For absolute values |·|1, |·|2 on k, TFAE:

  1. they are equivalent (same topology);
  2. same convergent sequences: xn → a in one ⟺ in the other;
  3. same "small" elements: |x|1 < 1 ⟺ |x|2 < 1;
  4. there is a real α > 0 with |x|1 = |x|2α for all x.

Plain-language: "Same topology" sounds abstract, but it secretly means something very concrete: one ruler is just a fixed power of the other (condition iv). Raising every size to the power α reshapes the numbers but preserves which ones are below 1 — and that's enough to keep all the topology identical. The hardest step is (iii)⇒(iv), worked out in Appendix B / Problem 72.

🔍 Real-World Example — "Same map, different scale"

Currency exchange / log scales: Suppose one stock tracker reports volatility as v and another reports it as v1.5. The raw numbers differ, but every statement of the form "this asset is more volatile than that one" or "volatility is shrinking toward zero" is identical in both systems. Raising to a positive power never changes the ordering or what tends to 0 — exactly condition (iv) of equivalence.

Richter vs. raw amplitude: Earthquake size in raw seismic amplitude vs. the (logarithmic) Richter scale are different numbers, yet "bigger quake" and "sequence of quakes dying down" mean the same in both. Equivalent absolute values are the algebraic version of this idea.

🟢 Theorem 3.1.4 — Ostrowski's Theorem Theorem

Statement: Every non-trivial absolute value on ℚ is equivalent to one of the |·|p, where p is either a prime number or p = ∞.

Why it's stunning: There is no exotic, undiscovered way to measure size on ℚ. Every possibility collapses onto our familiar list. The primes — and the single point at infinity — completely account for all of ℚ's geometry.

Proof Skeleton (two cases)

Case A — archimedean. Let n0 be the smallest positive integer with |n0| > 1; pick α with |n0| = n0α. Writing any integer in base n0 and applying a clever "take N-th roots, let N→∞" trick squeezes |n| between nα from both sides, giving |n| = nα. So |·| ∼ |·|∞.

Case B — non-archimedean. Now |n| ≤ 1 for all integers. Let n0 be the smallest integer with |n0| < 1. It must be prime (if n0 = ab with both factors smaller, both would have size 1, contradiction). Call it p. One checks |n| = 1 whenever p∤n, so writing n = pvn′ gives |n| = |p|v = c−v — equivalent to |·|p.

🔍 Real-World Example — A "fingerprint" classifier

Ostrowski is like proving that every possible fingerprint scanner, no matter how it's engineered, can only ever report one of a known finite catalog of fingerprints. In data terms: there's a complete classification. If someone hands you a mystery absolute value on ℚ, you don't need to invent new theory — you just run the two-case test (archimedean? find the smallest "big" integer; non-archimedean? find the smallest "shrinking" integer, and it'll be a prime) and read off the answer.

📐 Proposition 3.1.5 — The Product Formula Prop

Statement: For any x ∈ ℚ×,

∏p ≤ ∞ |x|p = 1,

where the product runs over all primes p together with the infinite prime ∞.

Plain-language: Each prime's absolute value sees only its own piece of the factorization; the absolute value at infinity sees the overall magnitude. Multiply them all and everything cancels perfectly to 1. It's a conservation law: the "information" in a rational number is shared among the primes, and the books always balance.

Proof Idea

Reduce to a positive integer x = p1a1···pkak. Then |x|q = 1 for primes q not in the factorization, |x|pi = pi−ai, and |x|∞ = p1a1···pkak. The finite factors exactly cancel the infinite one.

🔍 Real-World Example — Conservation & "fill in the missing reading"

Mass/energy balance: Like a conservation law in physics, the product formula says nothing is lost — the contributions from every prime and from infinity always multiply to exactly 1.

Error-correction / redundancy: Because the product is forced to be 1, if you know all the absolute values of x except one, you can recover the missing one. This redundancy is the backbone of the theory of heights used in modern Diophantine geometry (and conceptually similar to a checksum).

§3.2 Completions

We now build, for each prime p, the field ℚp. The idea mirrors how ℝ is built from ℚ: fill in the missing limits of Cauchy sequences. The twist is that "missing limits" are added by a clever algebraic trick — taking a quotient of the ring of all Cauchy sequences.

📘 Definition 3.2.1 — Cauchy, Complete, Dense

(i) Cauchy sequence: (xn) is Cauchy if for every ε > 0 there is an M with |xn − xm| < ε whenever m,n ≥ M.

(ii) Complete: k is complete if every Cauchy sequence in k has a limit in k.

(iii) Dense: S ⊂ k is dense if every open ball around every point of k contains a point of S.

Plain-language: A Cauchy sequence is one whose terms "bunch up" — they ought to converge because they're crowding into smaller and smaller balls. A field is complete if every sequence that ought to converge actually does. Dense means you can approximate anything in the big field arbitrarily well by something from the small subset (like approximating any real by a decimal).

📒 Lemma 3.2.2 — The Non-archimedean Cauchy Shortcut Lemma

Statement: In a non-archimedean field, (xn) is Cauchy if and only if limn→∞ |xn+1 − xn| = 0.

Plain-language: Normally "consecutive terms get close" is weaker than Cauchy (the harmonic series partial sums are a classic counterexample). But the ultrametric inequality is so strong that here the two conditions coincide. This makes p-adic analysis dramatically simpler: to check convergence of a series, you just check that its terms go to 0!

Proof Idea

For m = n+r, write xm − xn as a telescoping sum of consecutive differences. The ultrametric inequality bounds the whole sum by the maximum consecutive difference. If consecutive differences → 0, that maximum → 0 too.

🔍 Real-World Example — "Settling down" signals

GPS / iterative solvers: A numerical method (Newton's method, gradient descent) produces a sequence of estimates. It "ought to converge" when successive estimates barely change. In ordinary arithmetic that's not quite enough — but in the p-adic world, "successive estimates stop changing" is exactly convergence. This is why p-adic Hensel lifting (next chapter) is so clean.

📒 Lemma 3.2.3 — ℚ Has Holes Lemma

Statement: ℚ is not complete with respect to any of its non-trivial absolute values.

Plain-language: Everyone knows ℚ has a hole where √2 should be (the usual absolute value). The lemma says the p-adic worlds have holes too. For an odd prime p, pick a that is a square mod p but not a perfect square in ℚ; you can build a Cauchy sequence of better and better approximate square roots of a — but its "limit" √a doesn't live in ℚ. So a completion is genuinely needed.

Proof Idea (odd p)

Choose x0 with x02 ≡ a (mod p), then lift to xn with xn ≡ xn−1 (mod pn) and xn2 ≡ a (mod pn+1). Then |xn+1 − xn|p ≤ p−(n+1) → 0 (Cauchy), and |xn2 − a|p → 0, so any limit would be a square root of a — impossible in ℚ.

🔁 Interactive: Building √a in ℚp (a hole in ℚ)

Pick a prime and a target a. The tool lifts an approximate square root step by step (this is Hensel lifting!) and shows that xn2 → a p-adically, even though no rational √a exists.

Constructing ℚp: The Algebraic Machine

The construction has four moving parts. Here's the assembly line:

📘 Definition 3.2.4 — The Ring 𝒞 of Cauchy Sequences

𝒞 = 𝒞p(ℚ) is the set of all Cauchy sequences of rationals (w.r.t. |·|p).

Plain-language: Instead of magically conjuring the missing limits, we keep the sequences themselves as our raw material. The limit we don't have will be represented by the sequence we do have.

📐 Proposition 3.2.5 — 𝒞 is a Ring Prop

Termwise addition (xn)+(yn) = (xn+yn) and multiplication (xn)·(yn) = (xnyn) make 𝒞 a commutative ring with unity. (The only real check: sums/products of Cauchy sequences are Cauchy.)

📒 Lemma 3.2.6 — ℚ Sits Inside 𝒞 Lemma

The map x ↦ x̃ (the constant sequence x, x, x, …) is an injective ring homomorphism ℚ → 𝒞. So our raw material already contains a faithful copy of ℚ.

📘 Definition 3.2.7 — The Ideal 𝒩 of Null Sequences

𝒩 = {(xn) : xn → 0} = {(xn) : lim |xn|p = 0}, an ideal of 𝒞.

Plain-language: Two Cauchy sequences "ought" to have the same limit precisely when their difference dies out to 0. Those die-out sequences form 𝒩. Quotienting by 𝒩 is how we glue together sequences that should share a limit.

📒 Lemma 3.2.8 — 𝒩 is Maximal Lemma

Statement: 𝒩 is a maximal ideal of 𝒞.

Why we care: Quotienting a commutative ring by a maximal ideal yields a field. That's how ℚp ends up being a field and not just a ring.

Proof Idea

Take a Cauchy (xn) ∉ 𝒩. It is eventually bounded away from 0, so define yn = 1/xn past that point. Then (yn) is Cauchy and (xn)(yn) is "0…0,1,1,1,…", which differs from the constant 1̃ by a null sequence. Hence the ideal generated by (xn) and 𝒩 contains 1̃ — so it's all of 𝒞, proving maximality.

📘 Definition 3.2.9 — The Field ℚp
ℚp = 𝒞 / 𝒩

The field of p-adic numbers is the quotient of the ring of Cauchy sequences by the maximal ideal of null sequences. ℚ still embeds via x ↦ [x̃].

📒 Lemma 3.2.10 — Absolute Value Stabilizes Lemma

If (xn) ∈ 𝒞 is not null, the real numbers |xn|p are eventually constant. (Proof: once terms are bounded below by c and within c of each other, "all triangles are isosceles" forces |xn| = |xm|.) This makes the next definition well-posed.

📘 Definition 3.2.11 — Absolute Value on ℚp

For λ ∈ ℚp represented by a Cauchy sequence (xn), define |λ|p = limn→∞ |xn|p. Because of Lemma 3.2.10 this limit exists and doesn't depend on the chosen representative. Its image is still {pn : n ∈ ℤ} ∪ {0}.

📐 Proposition 3.2.12 — ℚ is Dense in ℚp Prop

Every p-adic number can be approximated as closely as you like by an ordinary rational. (Proof: represent λ by Cauchy (xn); the constant sequence x̃N for large N lands inside B(λ, ε).)

🟢 Theorem 3.2.13 — ℚp is Complete Theorem

Statement: Every Cauchy sequence in ℚp converges to an element of ℚp.

The subtlety: a Cauchy sequence in ℚp is a "sequence of (equivalence classes of) sequences." The trick is density: replace each λi by a nearby rational yi, show (yn) is itself Cauchy in ℚ (hence defines some λ ∈ ℚp), and prove λn → λ.

🟢 Theorem 3.2.14 — Existence & Uniqueness of ℚp Theorem

For each prime p there is a field ℚp with a non-archimedean |·|p such that: (i) ℚ ↪ ℚp induces the p-adic absolute value; (ii) ℚ is dense in ℚp; (iii) ℚp is complete. Moreover ℚp is unique up to unique isomorphism preserving absolute values.

Why this matters: the strong uniqueness lets us forget the messy Cauchy-sequence construction forever. From now on we work only with the three defining properties — just as we use ℝ without ever recalling Dedekind cuts.

🔍 Real-World Example — "Defined by its specs, not its wiring"

Engineering interfaces: Once a chip is "unique up to unique isomorphism" for a spec, any two implementations are interchangeable — you program against the interface (the three properties), never the internal wiring (Cauchy sequences). The reals work the same way: you use ℝ via completeness/density, not Dedekind cuts.

The punchline picture: ℝ "fills the number line." ℚp instead fills a totally disconnected, tree-like space — there's no line to picture, which is exactly why the p-adic world feels so different.
