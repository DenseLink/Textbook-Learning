Introduction

Chapter 4 — Exploring ℚₚ

Having constructed the p-adic numbers ℚₚ as the completion of ℚ with respect to the p-adic absolute value, we now explore their structure concretely. This chapter reveals how p-adic numbers can be represented as infinite base-p expansions (analogous to decimals, but extending infinitely to the left), introduces the ring of p-adic integers Zₚ, and presents Hensel's Lemma—a powerful tool for lifting solutions from modular arithmetic to the p-adics.

Roadmap

  • §4.2 — The p-adic integers Zₚ = {x ∈ ℚₚ : |x|ₚ ≤ 1} (compact valuation ring).
  • §4.3 — p-adic expansions: every x ∈ ℚₚ as Σ bₙ pⁿ (unique base-p series).
  • §4.5 — Hensel's Lemma: lifting roots from mod p to Zₚ.

4.1 — What We Already Know

From the construction in Chapter 3, we know ℚₚ is a complete field with a non-archimedean absolute value. Let's summarize key properties before exploring further.

Lemma 4.1.1

For each x ∈ ℚₚ, x ≠ 0, there exists an integer n ∈ ℤ such that |x|ₚ = p⁻ⁿ.

In other words, the image of |·|ₚ on ℚₚ is {pⁿ | n ∈ ℤ} ∪ {0}.

Lemma 4.1.2 (Extended Valuation)

The p-adic valuation vₚ extends from ℚ to ℚₚ: for x ∈ ℚₚ, x ≠ 0,

|x|ₚ = p⁻ᵛᵖ⁽ˣ⁾

We set vₚ(0) = +∞ by convention.

4.2 — p-adic Integers Zₚ

The p-adic integers are the "unit ball" in ℚₚ. They form a compact valuation ring.

Definition 4.2.1

The ring of p-adic integers is

Zₚ = {x ∈ ℚₚ : |x|ₚ ≤ 1}
Proposition 4.2.2 (Structure)

Zₚ is a local ring with unique maximal ideal pZₚ = {x : |x|ₚ < 1}.

Density: Given x ∈ Zₚ and n ≥ 1, there exists a unique integer α with 0 ≤ α ≤ pⁿ - 1 such that |x - α|ₚ ≤ p⁻ⁿ.

Every element of Zₚ is the limit of a unique coherent sequence (αₙ) where αₙ ≡ αₙ₋₁ (mod pⁿ⁻¹).

Corollary 4.2.7

Zₚ is compact, and ℚₚ is locally compact.

4.3 — The Elements of ℚₚ (p-adic Expansions)

Every p-adic number can be written uniquely as a base-p "decimal" expansion extending infinitely to the left.

Theorem 4.3.1 (p-adic Expansion)

Every x ∈ ℚₚ has a unique representation as

x = b₋ₘ p⁻ᵐ + b₋ₘ₊₁ p⁻ᵐ⁺¹ + ⋯ + b₋₁ p⁻¹ + b₀ + b₁ p + b₂ p² + ⋯ = Σ (n ≥ -m) bₙ pⁿ

with digits bₙ ∈ {0, 1, ..., p-1} and -m = vₚ(x).

Example: 1/(1-p) in ℚₚ

The geometric series 1 + p + p² + p³ + ⋯ converges in ℚₚ (since |p|ₚ < 1, terms → 0).

Multiplying by (1 - p): (1 - p)(1 + p + p² + ⋯) = 1.

Hence 1/(1-p) = 1 + p + p² + ⋯ (all digits = 1).

Example: -1 in ℚₚ

Write -1 = (p-1) + (p-1)p + (p-1)p² + ⋯ (all digits = p-1).

Check: this series converges (terms → 0), and equals (p-1)/(1-p) = -1.

4.5 — Hensel's Lemma

Hensel's Lemma is the p-adic analogue of Newton's method: roots modulo p lift uniquely to roots in Zₚ if the derivative is nonzero.

Theorem 4.5.1 (Hensel's Lemma)

Let f(X) ∈ Zₚ[X] and suppose α ∈ Zₚ satisfies:

  1. f(α) ≡ 0 (mod p) (α is a root mod p)
  2. f'(α) ≢ 0 (mod p) (simple root mod p)

Then there exists a unique root a ∈ Zₚ of f with a ≡ α (mod p).

Hensel Lifting: Construct a via Newton iteration: αₙ₊₁ = αₙ - f(αₙ)/f'(αₙ).

Example: √2 in ℚ₇

Does √2 exist in ℚ₇? Let f(X) = X² - 2. We need a root α ∈ ℤ/7ℤ of f. Check: 3² = 9 ≡ 2 (mod 7). So α = 3 works.

f'(X) = 2X, f'(3) = 6 ≢ 0 (mod 7). Hensel's Lemma applies! ∴ √2 exists in Z₇.
