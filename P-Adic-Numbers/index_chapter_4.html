Introduction

Chapter 4 — Exploring ℚₚ

Having constructed the p-adic numbers ℚₚ as the completion of ℚ with respect to the p-adic absolute value, we now explore their structure concretely. This chapter reveals how p-adic numbers can be represented as infinite base-p expansions (analogous to decimals, but extending infinitely to the left), introduces the ring of p-adic integers Zₚ, and presents Hensel's Lemma—a powerful tool for lifting solutions from modular arithmetic to the p-adics.

4.1 — What We Already Know

From the construction in Chapter 3, we know ℚₚ is a complete field with a non-archimedean absolute value. Let's summarize key properties before exploring further.

Lemma 4.1.1

For each x ∈ ℚₚ, x ≠ 0, there exists an integer n ∈ ℤ such that |x|ₚ = p⁻ⁿ.

In other words, the image of |·|ₚ on ℚₚ is {pⁿ | n ∈ ℤ} ∪ {0}.

Lemma 4.1.2 (Extended Valuation)

The p-adic valuation vₚ extends from ℚ to ℚₚ: for x ∈ ℚₚ, x ≠ 0,

|x|ₚ = p⁻ᵛᵖ⁽ˣ⁾

We set vₚ(0) = +∞ by convention.

4.2 — p-adic Integers Zₚ

The p-adic integers are the "unit ball" in ℚₚ. They form a compact valuation ring.

Definition 4.2.1

The ring of p-adic integers is

Zₚ = {x ∈ ℚₚ : |x|ₚ ≤ 1}
Proposition 4.2.2 (Structure)

Zₚ is a local ring with unique maximal ideal pZₚ = {x : |x|ₚ < 1}.

Density: Given x ∈ Zₚ and n ≥ 1, there exists a unique integer α with 0 ≤ α ≤ pⁿ - 1 such that |x - α|ₚ ≤ p⁻ⁿ.

Every element of Zₚ is the limit of a unique coherent sequence (αₙ) where αₙ ≡ αₙ₋₁ (mod pⁿ⁻¹).

Corollary 4.2.3 — ℤ is dense in Zₚ

Every p-adic integer is a limit of ordinary integers. (One could even define Zₚ as the completion of ℤ with respect to |·|ₚ.) The coherent sequence (αₙ) of Prop. 4.2.2(iii) is exactly the "coherent sequence" of Chapter 1.

Corollary 4.2.4 — ℚₚ = Zₚ[1/p]

For every x ∈ ℚₚ there is an n ≥ 0 with pⁿx ∈ Zₚ. Multiplication by p is a homeomorphism, and the sets pⁿZₚ (n ∈ ℤ) form a fundamental system of neighborhoods of 0 covering all of ℚₚ.

This gives a clean description of the valuation: vₚ(x) is the largest n with x ∈ pⁿZₚ. Crucially, |x − y|ₚ ≤ p⁻ⁿ ⟺ x − y ∈ pⁿZₚ, so B(0, p⁻ⁿ) = pⁿZₚ — topology and algebra are tightly linked.

Corollary 4.2.5 — Exact sequence & finite quotients

For each n ≥ 1 the sequence 0 → Zₚ —(·pⁿ)→ Zₚ → ℤ/pⁿℤ → 0 is exact, so

Zₚ / pⁿZₚ ≅ ℤ/pⁿℤ

The cosets a + pⁿZₚ (a = 0,…,pⁿ−1) are precisely the balls B(a, p⁻ⁿ). The finiteness of these quotients is exactly what forces Zₚ to be compact.

Corollary 4.2.6 — Topology of ℚₚ

ℚₚ is a totally disconnected, Hausdorff topological space: the connected component of any point is just that point, yet any two distinct points can be separated by disjoint (cl)open balls.

Corollary 4.2.7 — Compactness

Zₚ is compact, and ℚₚ is locally compact. (More generally: a complete field with principal valuation ideal is locally compact iff its residue field is finite.)

Definition — The p-adic units Zₚ×

The invertible elements of Zₚ form a group, the p-adic units:

Zₚ× = {x ∈ ℚₚ : |x|ₚ = 1}, Zₚ× ∩ ℚ = {a/b : p ∤ ab}

It is a closed (hence compact) subset of Zₚ. Every nonzero x ∈ ℚₚ factors uniquely as x = pᵛᵖ⁽ˣ⁾ · u with u ∈ Zₚ×.

4.3 — The Elements of ℚₚ (p-adic Expansions)

We give two concrete descriptions of elements of ℚₚ — both met in Chapter 1: coherent sequences (theoretical) and p-adic expansions (the most concrete).

Coherent Sequences & the Inverse Limit

Writing φₙ : Zₚ → ℤ/pⁿℤ for the projections and Aₙ = ℤ/pⁿℤ, a coherent sequence is a tuple (αₙ) with αₙ ∈ Aₙ and ψₙ(αₙ) = αₙ₋₁ (reduction mod pⁿ⁻¹).

Proposition 4.3.1: The maps φₙ identify Zₚ, as a topological ring, with the closed subring of ∏ₙ Aₙ consisting of coherent sequences. In high-brow language, Zₚ = lim← ℤ/pⁿℤ is the inverse limit of the finite rings ℤ/pⁿℤ — a purely algebraic construction with a universal property.

Theorem 4.3.3 — Corollary (p-adic Expansion)

Every x ∈ ℚₚ has a unique representation as

x = b₋ₘ p⁻ᵐ + b₋ₘ₊₁ p⁻ᵐ⁺¹ + ⋯ + b₋₁ p⁻¹ + b₀ + b₁ p + b₂ p² + ⋯ = Σ (n ≥ -m) bₙ pⁿ

with digits bₙ ∈ {0, 1, ..., p-1} and -m = vₚ(x).

Example: 1/(1-p) in ℚₚ

The geometric series 1 + p + p² + p³ + ⋯ converges in ℚₚ (since |p|ₚ < 1, terms → 0).

Multiplying by (1 - p): (1 - p)(1 + p + p² + ⋯) = 1.

Hence 1/(1-p) = 1 + p + p² + ⋯ (all digits = 1).

Example: -1 in ℚₚ

Write -1 = (p-1) + (p-1)p + (p-1)p² + ⋯ (all digits = p-1).

Check: this series converges (terms → 0), and equals (p-1)/(1-p) = -1.

Corollary 4.3.5 — Arbitrary digit sets

The digits need not be {0,…,p−1}. Any set A ⊂ Zₚ of representatives of ℤ/pℤ works: every x ∈ ℚₚ has a unique expansion Σ bₙ pⁿ with bₙ ∈ A. The elements of A need not even be integers — e.g. one may use 0 together with the (p−1)-st roots of unity (the Teichmüller representatives). The expansion depends strongly on the chosen digit set.

R vs ℚₚ — a clean analogy

A real x = Σ bᵢ10⁻ⁱ extends infinitely to the right (and is not unique: 0.999… = 1). A p-adic x = Σ bₙpⁿ extends infinitely to the left (and is unique). In both cases eventually-periodic expansions correspond exactly to rational numbers. Both ℝ and ℚₚ are locally compact, but ℚₚ is totally disconnected.

4.4 — What Does ℚₚ Look Like?

We have a beautiful mental picture of ℝ as a line. Can we "see" ℚₚ (or at least Zₚ)? The challenge: ℚₚ is totally disconnected but not discrete — its points are not connected to each other, yet none of them is isolated. We can't draw that with intervals or with scattered dots.

What "visualizing" means here

The real line encodes topology and order and metric. For Zₚ we settle for a picture that preserves the topology (which sequences converge to which limits) but not the metric. So distances in the picture won't be proportional to p-adic distances — but convergent sequences will still look convergent.

The Cantor middle-thirds set C

Start with C₀ = [0,1], remove the open middle third to get C₁ = [0,⅓] ∪ [⅔,1], repeat forever: C = ⋂ₙ Cₙ. The result is compact, totally disconnected, and "perfect" (no isolated points). A point lies in C iff it has a base-3 expansion using only digits {0, 2}.

Theorem 4.4.1 — Z₂ ≅ Cantor set

Z₂ with the 2-adic metric is homeomorphic to the middle-thirds Cantor set C ⊂ ℝ. The bijection sends a 2-adic expansion Σ bₙ2ⁿ (bₙ ∈ {0,1}) to the base-3 number Σ 2bₙ/3ⁿ⁺¹ (digits in {0,2}):

f(b₀ + b₁·2 + b₂·2² + ⋯) = 2b₀/3 + 2b₁/3² + 2b₂/3³ + ⋯

The hierarchical "split each piece into 2" structure of C mirrors "each residue mod 2ⁿ splits into 2 mod 2ⁿ⁺¹".

Theorem 4.4.2 — All Zₚ are homeomorphic

For any prime p, Z₂ ≅ Zₚ (topologically). This follows from a topology theorem: any compact, perfect, totally disconnected subset of ℝ is homeomorphic to the Cantor set. So every Zₚ is "a Cantor set."

Fractals via "vector digits"

For prettier pictures, place a point xᵢ ∈ ℝⁿ for each digit i and map Σ aᵢpⁱ ↦ Σ (1/bⁱ⁺¹) x_{aᵢ} for a large base b. The image of Zₚ becomes a self-similar fractal. Using the vertices of a regular p-gon, Sage's Zp(5).plot() draws such a picture; Z₃ comes out as the Sierpiński gasket. Scaling these copies by θ⁻¹ repeatedly builds a picture of all of ℚₚ.

Caveat

These pictures are topologically faithful but not metrically faithful, and they don't encode the algebraic (field) structure. Fun to look at — but limited as a working tool.

4.5 — Hensel's Lemma

Hensel's Lemma is the p-adic analogue of Newton's method: roots modulo p lift uniquely to roots in Zₚ if the derivative is nonzero.

Theorem 4.5.1 (Hensel's Lemma)

Let f(X) ∈ Zₚ[X] and suppose α ∈ Zₚ satisfies:

  1. f(α) ≡ 0 (mod p) (α is a root mod p)
  2. f'(α) ≢ 0 (mod p) (simple root mod p)

Then there exists a unique root a ∈ Zₚ of f with a ≡ α (mod p).

Hensel Lifting: Construct a via Newton iteration: αₙ₊₁ = αₙ - f(αₙ)/f'(αₙ).

Example: √2 in ℚ₇

Does √2 exist in ℚ₇? Let f(X) = X² - 2. We need a root α ∈ ℤ/7ℤ of f. Check: 3² = 9 ≡ 2 (mod 7). So α = 3 works.

f'(X) = 2X, f'(3) = 6 ≢ 0 (mod 7). Hensel's Lemma applies! ∴ √2 exists in Z₇.

Why Newton's method? (the lifting step)

To go from αₙ (good mod pⁿ) to αₙ₊₁ = αₙ + bpⁿ, expand F(αₙ + bpⁿ) ≡ F(αₙ) + F'(αₙ)·bpⁿ (mod pⁿ⁺¹). Writing F(αₙ) = pⁿx and dividing by pⁿ gives x + F'(αₙ)b ≡ 0 (mod p). Since F'(αₙ) is a unit (≢ 0 mod p), b is uniquely solvable. Substituting back yields exactly

αₙ₊₁ = αₙ − F(αₙ)/F'(αₙ)

— Newton's iteration. Unlike the real case, in Zₚ it always converges (provided F'(α₁) ≢ 0), never leaves Zₚ, and the limit is the unique root near α₁.

Theorem 4.5.3 — Stronger Hensel (Problem 120)

The hypotheses can be weakened: it suffices that

|F(α₁)|ₚ < |F'(α₁)|ₚ²

Then the Newton sequence still converges to a unique root α with |α − α₁|ₚ < |F'(α₁)|ₚ. This version handles cases where F'(α₁) is divisible by p — essential for p = 2 (e.g. lifting square roots, where F'(X) = 2X is always even).

4.6 — Using Hensel's Lemma

Two beautiful applications: determining exactly which roots of unity live in ℚₚ, and classifying the squares in ℚₚ.

Roots of Unity

An m-th root of unity ζ satisfies ζᵐ = 1; it is primitive if no smaller power equals 1. Any root of unity has |ζ|ₚ = 1, so ζ ∈ Zₚ×. Apply Hensel to F(X) = Xᵐ − 1, with F'(X) = mXᵐ⁻¹ (a unit mod p iff p ∤ m).

Proposition 4.6.1 — Roots of unity in ℚₚ

For p ∤ m, ℚₚ contains a primitive m-th root of unity if and only if m divides p − 1.

Consequently the roots of unity of order prime to p are exactly the (p−1)-st roots of unity: there are exactly p − 1 of them, forming a cyclic group ≅ ℤ/(p−1)ℤ. (For p² | order: no pⁿ-th roots exist except ±1 when p = 2.)

Example: 4th roots of unity in ℚ₅

p = 5, p − 1 = 4. Since 4 | 4, ℚ₅ contains a primitive 4th root of unity — i.e. a square root of −1! Indeed 2² = 4 ≡ −1 (mod 5), and Hensel lifts 2 to i ∈ Z₅.

Squares in ℚₚ

Proposition 4.6.2 & Corollary 4.6.3 (p ≠ 2)

For an odd prime p and a unit b ∈ Zₚ×: if α₁² ≡ b (mod p) has a solution, then b is a square in Zₚ× (apply Hensel to X² − b; 2α₁ ≢ 0 mod p).

In general, x ∈ ℚₚ is a square iff x = p²ⁿ y² with y ∈ Zₚ× — that is, vₚ(x) is even and the unit part is a QR mod p. The group ℚₚ×/(ℚₚ×)² has order 4, with coset representatives {1, p, c, cp} where c is a non-residue mod p. (Compare ℝ: ℝ×/(ℝ×)² = {1, −1}, order 2 — a "rule of signs.")

The case p = 2

Here F'(X) = 2X is always even, so we need the stronger Hensel (Thm 4.5.3). Result: a 2-adic unit b is a square iff b ≡ 1 (mod 8). The group ℚ₂×/(ℚ₂×)² has order 8, generated by −1, 5, 2; representatives {1, −1, 5, −5, 2, −2, 10, −10}.

4.7 — Hensel's Lemma for Polynomials

The first Hensel's Lemma lifts a root, i.e. a linear factor (X − α). The general version lifts any factorization mod p — provided the factors are coprime mod p.

Definition 4.7.1 — Relatively prime mod p

Polynomials g, h ∈ Zₚ[X] are relatively prime modulo p if their reductions ḡ, h̄ ∈ 𝔽ₚ[X] satisfy gcd(ḡ, h̄) = 1; equivalently, ∃ a, b ∈ Zₚ[X] with a·g + b·h ≡ 1 (mod p).

This generalizes "f'(α) ≢ 0 mod p" (which says (X − α) does not divide the cofactor mod p — i.e. α is not a double root).

Theorem 4.7.2 — Hensel's Lemma for Polynomials

Let f ∈ Zₚ[X], and suppose f ≡ g₁·h₁ (mod p) where g₁ is monic and g₁, h₁ are relatively prime mod p. Then there exist g, h ∈ Zₚ[X] with:

  • g monic, with g ≡ g₁ and h ≡ h₁ (mod p),
  • and a true factorization over Zₚ: f = g·h.
Proof idea (successive approximation)

Build gₙ, hₙ with f ≡ gₙhₙ (mod pⁿ), gₙ monic of fixed degree, gₙ₊₁ ≡ gₙ (mod pⁿ). To pass from level n to n+1, set g = gₙ + p·r, h = hₙ + p·s and solve k ≡ r·hₙ + s·gₙ (mod p) where pk = f − gₙhₙ. Coprimality gives a, b with a·gₙ + b·hₙ ≡ 1, so r̃ = b·k, s̃ = a·k work — then reduce r̃ mod gₙ to keep g monic of the right degree. Taking limits gives g, h over Zₚ. (Same engine as the first Hensel's Lemma.)

Example (Problem 126)

p = 2, f(X) = 2X² + X + 2. Mod 2 this is just X, so take g₁ = X, h₁ = 1 (coprime mod 2). Hensel lifts this to a genuine factorization of f over Z₂ — the trick of reducing r̃ mod g₁ is what keeps the monic factor degree-1.

4.8 — Local and Global

Deciding if a polynomial has roots in each ℚₚ (a local question) is easy via Hensel. The dream: patch local information together to answer the hard global question of roots in ℚ. This is the local-global (Hasse) principle.

"Local" vs "Global" — the convention p ≤ ∞

Following Hensel's analogy, each ℚₚ carries "local" information near the prime p, with ℝ = ℚ_∞ treated as the prime "at infinity." A root in ℚ ("global") is automatically a root in every ℚₚ ("local everywhere"). So: no local root at some p ⟹ no global root.

  • X² + 1 = 0: no root in ℝ ⟹ none in ℚ.
  • X² − 2 = 0: no root in ℚ₂ ⟹ none in ℚ.
Proposition 4.8.1 — Squares (a clean local-global)

x ∈ ℚ is a square if and only if it is a square in every ℚₚ, p ≤ ∞. (Square at ∞ ⟹ positive; square at each finite p ⟹ vₚ(x) even ⟹ x is a square in ℚ.)

Theorem 4.8.2 — Hasse–Minkowski

A quadratic form F(X₁,…,Xₙ) = Σ cᵢⱼXᵢXⱼ over ℚ has a non-trivial zero in ℚ if and only if it has a non-trivial zero in every ℚₚ, p ≤ ∞.

The local conditions are each decidable (Hensel + finitely many primes matter), so this completely solves the rational solvability of quadratic forms.

⚠️ The naïve principle is FALSE in general

"Solvable locally everywhere ⟺ solvable globally" fails beyond quadratics:

  • (X²−2)(X²−17)(X²−34) = 0 has a root in every ℚₚ but none in ℚ (Problem 129).
  • X⁴ − 17 = 2Y² is solvable locally everywhere but not in ℚ (Problem 130).

Yet the idea remains central — the Birch & Swinnerton-Dyer conjecture is a deep modern descendant.

Worked machinery: aX² + bY² + cZ² = 0

For pairwise-coprime square-free integers a, b, c, Hasse–Minkowski reduces to checking each ℚₚ. Chevalley–Warning (Prop 4.8.3) handles odd p ∤ abc automatically (there's always a non-trivial solution mod p, which Hensel lifts). The remaining conditions (Prop 4.8.6) for a non-trivial rational solution are:

  1. a, b, c are not all the same sign (the ℝ / ∞ condition);
  2. for each odd prime dividing a: −b/c is a QR mod p (∃ r with b + r²c ≡ 0 mod p); similarly for primes dividing b and c;
  3. if a, b, c all odd: some two of them sum to 0 mod 4;
  4. if one is even: b+c or a+b+c is divisible by 8 (sym.).
