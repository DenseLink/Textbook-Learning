4.4 вҖ” What Does в„ҡвӮҡ Look Like?

We have a beautiful mental picture of в„қ as a line. Can we "see" в„ҡвӮҡ (or at least ZвӮҡ)? The challenge: в„ҡвӮҡ is totally disconnected but not discrete вҖ” its points are not connected to each other, yet none of them is isolated. We can't draw that with intervals or with scattered dots.

4.5 вҖ” Hensel's Lemma

Hensel's Lemma is the p-adic analogue of Newton's method: roots modulo p lift uniquely to roots in ZвӮҡ if the derivative is nonzero.

Theorem 4.5.1 (Hensel's Lemma)

Let f(X) вҲҲ ZвӮҡ[X] and suppose Оұ вҲҲ ZвӮҡ satisfies:

  1. f(Оұ) вүЎ 0 (mod p) (Оұ is a root mod p)

Example: вҲҡ2 in в„ҡвӮҮ

Does вҲҡ2 exist in в„ҡвӮҮ? Let f(X) = XВІ - 2. We need a root Оұ вҲҲ в„Ө/7в„Ө of f. Check: 3ВІ = 9 вүЎ 2 (mod 7). So Оұ = 3 works.

f'(X) = 2X, f'(3) = 6 вүў 0 (mod 7). Hensel's Lemma applies! вҲҙ вҲҡ2 exists in ZвӮҮ.

Why Newton's method? (the lifting step)

To go from ОұвӮҷ (good mod pвҒҝ) to ОұвӮҷвӮҠвӮҒ = ОұвӮҷ + bpвҒҝ, expand F(ОұвӮҷ + bpвҒҝ) вүЎ F(ОұвӮҷ) + F'(ОұвӮҷ)В·bpвҒҝ (mod pвҒҝвҒәВ№). Writing F(ОұвӮҷ) = pвҒҝx and dividing by pвҒҝ gives x + F'(ОұвӮҷ)b вүЎ 0 (mod p). Since F'(ОұвӮҷ) is a unit (вүў 0 mod p), b is uniquely solvable. Substituting back yields exactly

ОұвӮҷвӮҠвӮҒ = ОұвӮҷ вҲ’ F(ОұвӮҷ)/F'(ОұвӮҷ)

вҖ” Newton's iteration. Unlike the real case, in ZвӮҡ it always converges (provided F'(ОұвӮҒ) вүў 0), never leaves ZвӮҡ, and the limit is the unique root near ОұвӮҒ.

The hypotheses can be weakened: it suffices that

|F(ОұвӮҒ)|вӮҡ < |F'(ОұвӮҒ)|вӮҡВІ

Then the Newton sequence still converges to a unique root Оұ with |Оұ вҲ’ ОұвӮҒ|вӮҡ < |F'(ОұвӮҒ)|вӮҡ. This version handles cases where F'(ОұвӮҒ) is divisible by p вҖ” essential for p = 2 (e.g. lifting square roots, where F'(X) = 2X is always even).

4.7 вҖ” Hensel's Lemma for Polynomials

The first Hensel's Lemma lifts a root, i.e. a linear factor (X вҲ’ Оұ). The general version lifts any factorization mod p вҖ” provided the factors are coprime mod p.

4.8 вҖ” Local and Global

Deciding if a polynomial has roots in each в„ҡвӮҡ (a local question) is easy via Hensel. The dream: patch local information together to answer the hard global question of roots in в„ҡ. This is the local-global (Hasse) principle.

"Local" vs "Global" вҖ” the convention p вүӨ вҲһ

Following Hensel's analogy, each в„ҡвӮҡ carries "local" information near the prime p, with в„қ = в„ҡ_вҲһ treated as the prime "at infinity." A root in в„ҡ ("global") is automatically a root in every в„ҡвӮҡ ("local everywhere"). So: no local root at some p вҹ№ no global root.

  • XВІ + 1 = 0: no root in в„қ вҹ№ none in в„ҡ.
  • XВІ вҲ’ 2 = 0: no root in в„ҡвӮӮ вҹ№ none in в„ҡ.
Proposition 4.8.1 вҖ” Squares (a clean local-global)

x вҲҲ в„ҡ is a square if and only if it is a square in every в„ҡвӮҡ, p вүӨ вҲһ. (Square at вҲһ вҹ№ positive; square at each finite p вҹ№ vвӮҡ(x) even вҹ№ x is a square in в„ҡ.)

A quadratic form F(XвӮҒ,вҖҰ,XвӮҷ) = ОЈ cбөўвұјXбөўXвұј over в„ҡ has a non-trivial zero in в„ҡ if and only if it has a non-trivial zero in every в„ҡвӮҡ, p вүӨ вҲһ.

The local conditions are each decidable (Hensel + finitely many primes matter), so this completely solves the rational solvability of quadratic forms.
