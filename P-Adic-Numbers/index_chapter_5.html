Introduction

Chapter 5 — Elementary Analysis in ℚₚ

Just as we study calculus and analysis in ℝ, we can develop p-adic analysis in ℚₚ. The ultrametric property makes many things simpler than in real analysis: a series converges if and only if its terms tend to zero (no comparison test needed!). We explore sequences, power series, the p-adic exponential and logarithm, and the structure of the unit group Zₚ×.

5.1 — Sequences and Series

ℚₚ is complete, so every Cauchy sequence converges. The ultrametric property drastically simplifies convergence tests.

Lemma 5.1.1 (Cauchy Criterion)

A sequence (aₙ) in ℚₚ is Cauchy ⟺ lim |aₙ₊₁ - aₙ|ₚ = 0.

(In ℝ, must check all pairs; in ℚₚ, consecutive terms suffice!)

Corollary 5.1.2 (THE Series Test)

Σaₙ converges in ℚₚ ⟺ aₙ → 0.

Moreover, if Σaₙ converges, then

|Σaₙ|ₚ ≤ max |aₙ|ₚ

(The sum is no bigger than the largest term!)

Contrast with ℝ

In ℝ, aₙ → 0 is necessary but not sufficient for Σaₙ to converge (e.g., Σ 1/n diverges).

In ℚₚ, it's both necessary and sufficient. Convergence is much easier to check!

5.2 — Functions, Continuity, Derivatives

Functions on ℚₚ behave similarly to functions on ℝ in many ways, but with surprising twists due to the ultrametric topology.

Definition 5.2.1 (Continuity)

f: U → ℚₚ is continuous at a if ∀ε>0 ∃δ>0: |x-a|ₚ<δ ⟹ |f(x)-f(a)|ₚ<ε.

f is uniformly continuous on U if δ depends only on ε, not on the point.

Locally Constant Functions

Every function f: Zₚ → ℚₚ that is locally constant (constant on cosets of pⁿZₚ for some n) is continuous.

Example: f(x) = x mod pⁿ is locally constant.

Derivatives

The derivative of f at a is f'(a) = lim_{h→0} [f(a+h)-f(a)]/h, if the limit exists in ℚₚ.

Standard rules: sum, product, quotient, chain rule all hold (same proofs as in ℝ).

Polynomial Derivatives

For f(x) = xⁿ, we have f'(x) = nxⁿ⁻¹ just as in real analysis.

But beware: if p | n, the derivative may vanish in unexpected ways!

5.3 — Integrals

Integration in ℚₚ uses Haar measure on the compact groups Zₚ and its cosets, giving a natural notion of "uniform distribution."

Haar Measure on Zₚ

There exists a unique (up to scalar) translation-invariant measure μ on Zₚ with μ(Zₚ) = 1.

Key property: μ(a + pⁿZₚ) = p⁻ⁿ (each coset of pⁿZₚ has measure p⁻ⁿ).

Integration Formula

For continuous f: Zₚ → ℚₚ, the integral

∫_{Zₚ} f dμ

can be computed as the limit of Riemann sums over partitions into cosets of pⁿZₚ.

Example: ∫ xᵏ dμ

For k ≥ 0, ∫_{Zₚ} xᵏ dμ can be computed using symmetry.

If p∤(k+1), the integral is 0 (by averaging over cosets). If p | (k+1), more subtle analysis is needed.

5.4 — Power Series

Power series f(X) = Σaₙ Xⁿ in ℚₚ have a radius of convergence r, determined by the Cauchy-Hadamard formula.

Definition (Radius of Convergence)

For f(X) = Σaₙ Xⁿ, the radius of convergence is

1/r = lim sup |aₙ|ₚ^(1/n)

f(x) converges if |x|ₚ < r, diverges if |x|ₚ > r.

Example: Geometric Series

1/(1 - pX) = 1 + pX + p²X² + ⋯ converges for |X|ₚ < 1/p (i.e., |pX|ₚ < 1).

Strassman's Theorem 5.6.1

Let f(X) = Σaₙ Xⁿ converge on |X|ₚ ≤ r. If not all aₙ ≡ 0 (mod p), let N = max{n : |aₙ|ₚ rⁿ is maximal}.

Then f has at most N zeros in {x : |x|ₚ ≤ r}.

(Contrast: in ℂ, nonconstant power series have infinitely many zeros!)

5.5 — Functions Defined by Power Series

Power series convergent on a ball define analytic functions, which can be differentiated term-by-term.

Theorem 5.5.1 (Term-by-Term Differentiation)

If f(x) = Σaₙxⁿ has radius r > 0, then f is differentiable on |x|ₚ < r, and

f'(x) = Σ naₙxⁿ⁻¹

with the same radius of convergence r.

Theorem 5.5.3 (Uniqueness)

If two power series Σaₙxⁿ and Σbₙxⁿ converge to the same function on a ball, then aₙ = bₙ for all n.

(Power series representations are unique.)

Algebraic Operations

If f = Σaₙxⁿ and g = Σbₙxⁿ both converge on |x|ₚ < r, then:

  • Sum: f+g = Σ(aₙ+bₙ)xⁿ
  • Product: fg = Σ(Σᵢ₊ⱼ₌ₙ aᵢbⱼ)xⁿ (Cauchy product)
  • Composition: f(g(x)) can be computed (if |g(x)|ₚ small enough)

5.6 — Strassman's Theorem

A remarkable result: nonconstant power series in ℚₚ have finitely many zeros (unlike in ℂ!).

Theorem 5.6.1 (Strassman)

Let f(x) = Σaₙxⁿ converge on the closed ball |x|ₚ ≤ r.

Suppose not all aₙ ≡ 0 (mod p). Define

N = max{n : |aₙ|ₚ rⁿ = max_k |aₖ|ₚ rᵏ}

Then f has at most N zeros in {x : |x|ₚ ≤ r}.

Proof Sketch

If x₀,...,xₙ are N+1 distinct zeros with |xᵢ|ₚ ≤ r, write f(x) = (x-x₀)⋯(x-xₙ)g(x).

Analyze |aₖ|ₚ rᵏ using the factorization; the ultrametric inequality on coefficients forces |aₙ|ₚ rᴺ to not be maximal — contradiction.

Contrast with ℂ

In ℂ, sin(x) = x - x³/6 + ⋯ has infinitely many zeros (πℤ).

In ℚₚ, the p-adic analogue has at most finitely many zeros in any ball!

5.7 — Exponential and Logarithm

The p-adic exponential and logarithm are defined by familiar power series, but have restricted domains (not global homomorphisms like in ℂ).

p-adic Exponential

exp(x) = Σ xⁿ/n!

Converges for:

  • p > 2 (odd): |x|ₚ < p^(-1/(p-1))
  • p = 2: |x|₂ < 1/4

Properties: exp(0) = 1; exp(x+y) = exp(x)·exp(y) only on restricted domain.

p-adic Logarithm

log(1+x) = Σ (-1)^(n-1) xⁿ/n

Converges for:

  • p ≠ 2: |x|ₚ < 1
  • p = 2: |x|₂ < 1/2

Properties: log(1) = 0; log((1+x)(1+y)) = log(1+x) + log(1+y) on appropriate domain.

5.8 — The Structure of Zₚ×

The unit group Zₚ× = {x ∈ Zₚ : |x|ₚ = 1} has a beautiful structure involving roots of unity.

Theorem 5.8.1

For p odd:

Zₚ× ≅ μ_{p-1} × (1 + pZₚ)

where μ_{p-1} = {ζ : ζ^(p-1) = 1} (roots of unity), and 1 + pZₚ ≅ Zₚ (via log).

For p = 2:

Z₂× ≅ {±1} × (1 + 4Z₂)

(Z₂× is not cyclic, unlike Zₚ× for odd p.)

Example: Z₅×

μ₄ = {1, 2, 3, 4} (mod 5 roots of unity, cyclic of order 4).

1 + 5Z₅ = {1 + 5a : a ∈ Z₅} ≅ Z₅ (principal units).

Z₅× ≅ ℤ/4ℤ × Z₅.

5.9 — The Binomial Series

The binomial series (1+x)ᵃ for arbitrary a ∈ ℤₚ converges in ℚₚ under suitable conditions.

Binomial Coefficient (p-adic)

For a ∈ ℤₚ and n ∈ ℕ, define

(a choose n) = a(a-1)⋯(a-n+1) / n!

This is well-defined in ℚₚ (denominator n! may have p-divisibility, but the product is in Zₚ for suitable a).

Theorem 5.9.1 (Binomial Series)

For a ∈ ℤₚ and |x|ₚ < 1, the series

(1+x)ᵃ = Σ (a choose n) xⁿ

converges in ℚₚ.

Key properties:

  • (1+x)ᵃ(1+x)ᵇ = (1+x)ᵃ⁺ᵇ (on domain |x|ₚ < 1)
  • Extends the usual binomial theorem from finite a ∈ ℕ to all a ∈ ℤₚ
Example: (1+x)⁻¹

Taking a = -1:

(1+x)⁻¹ = 1 - x + x² - x³ + ⋯

(Geometric series! Converges for |x|ₚ < 1.)

5.10 — Interpolation

Mahler's Theorem: Every continuous function f: ℤₚ → ℚₚ can be written as a convergent series in binomial polynomials.

Theorem 5.10.1 (Mahler)

For any continuous f: ℤₚ → ℚₚ, there exist unique aₙ ∈ ℚₚ with aₙ → 0 such that

f(x) = Σ aₙ (x choose n)

for all x ∈ ℤₚ, where the series converges uniformly.

Proof Idea

Use the fact that f is uniformly continuous on the compact space Zₚ.

Approximate f by step functions constant on cosets of pⁿZₚ, then express those via binomial basis.

The binomial polynomials {(x choose n)} form an orthogonal basis with respect to the Haar measure.

Example: Polynomials

If f(x) = xᵏ, then f can be written as

xᵏ = Σ Stirling(k,n) (x choose n)

where Stirling(k,n) are Stirling numbers of the second kind.
